Autores: Jairo González, Alvaro Lopez, Nicolas Rodrigo.

En la carpeta src se encuentra el código correspondiente a los primeros 6 apartados.

En la carpeta HuffmanDodes se encuentra el código proporcionado junto a una nueva clase llamada MainExperimentos,
que realiza los experimentos pedidos utilizando el código proporcionado. Estos experimentos calculan la compresion
media de strings generados aleatoriamente. 

El primer experimento utiliza una longitud de String fija y el segundo experimento utiliza una longitud variable.
Los parámetros de dichos experimentos se encuentran al principio de la clase y se pueden modificar para comprobar
distintos rangos de longitudes y de número de Strings generados.
